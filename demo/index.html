The select-a11y Web Component:
An Accessible Custom Element for Dropdown Selection

Jonathan Pool

Revised 28 May 2018

Introduction

This page demonstrates a prototype of the select-a11y custom element.

This test element distinguishes itself from the built-in select element by setting its size attribute to 5. Therefore, when the page includes a select element whose is attribute has the value select-a11y and which does not have a size attribute, 5 of the option elements inside the select element should be visible. The same select element would otherwise (i.e. without is="select-a11y") make only 1 or 4 of its option elements visible.
